The **Windows** workstream covers installing the Swift toolchain on
Windows, integrating with the Visual Studio runtime, and using Swift
Package Manager to build native applications and libraries on Windows.

## Topics

- Installing the official Windows Swift toolchain
- Linking against Win32 and the C runtime
- Cross-compiling Swift code for Windows from CI

## From swift.org

Windows is an officially-supported Swift platform with its own
workgroup and installer story:

- [Install Swift on Windows](https://www.swift.org/install/windows/) —
  the official installation hub, with detailed instructions for
  [winget](https://www.swift.org/install/windows/winget/),
  [Scoop](https://www.swift.org/install/windows/scoop/), and a
  [manual installer](https://www.swift.org/install/windows/manual/).
- [Swift Windows Workgroup](https://www.swift.org/windows-workgroup/)
  — charter and ongoing work to improve Swift's Windows experience.
- [Swift platform support](https://www.swift.org/platform-support/) —
  the support matrix and toolchain availability for Windows targets.
