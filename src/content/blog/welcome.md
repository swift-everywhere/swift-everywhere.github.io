Swift Everywhere is dedicated to making Swift a great choice for *every*
platform — Android, server, embedded, WebAssembly, and Windows. This site
is now a static Astro project, which lets us publish blog posts and
project landing pages alongside the existing Android build status report.

Browse the [projects](/projects/) page for an overview of each
workstream, or jump straight into the [blog](/blog/).
